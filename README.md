# FitTrack

A modern fitness tracking Android app with premium features.

## Features

- Track workout sessions with duration and calories
- Multiple workout types (Running, Walking, Cycling, etc.)
- Real-time workout tracking
- Basic workout statistics
- Dark mode support
- Premium features (analytics, personalized plans)

## Tech Stack

- Kotlin
- MVVM Architecture
- Hilt for DI
- Room Database
- Material Design
- Google Play Billing

## Setup

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle files
4. Run on emulator/device
